What is grid based design?

Grid based design follows a grid design pattern, which our eyes are naturally drawn to. Grid based design provides for a clean user experience, and so nearly all websites follow this pattern.

Whats all the hype about responsive webpages?

Responsibe webpages are cool because they allow users to have the same experience of the website regardless of how they are viewing the content - whether it be ipad,laptop, or phone. The webpage responds to the device.
