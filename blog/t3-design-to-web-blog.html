What is mobile first design and why it's important?

The mobile-first approach is exactly as it sounds: designing for the smallest screen and working your way up. It is one of the best strategies to create either a responsive or adaptive design.

What are frameworks, what are their pros and cons?

A CSS framework is a pre-prepared software framework that is meant to allow for easier, more standards-compliant web design using the Cascading Style Sheets language. Most of these frameworks contain at least a grid. Pros of using a framework include:

  • Speeds up development
  • Enables cross border functionality
  • Gives clean, symetrical layouts
  • Enforces good web-design habits

However, the cons include:

  • Restricted freedom
  • Adds extra code
  • Forces you to use the frameworks semantics
